M-Logger ファームウェア更新キット
=================================

【内容物】
  update.bat                      ダブルクリックして更新を開始
  avrdude.exe                     書き込みツール (avrdude、GPL ライセンス)
  avrdude.conf                    avrdude 設定ファイル
  mlogger_main.X.production.hex   新ファームウェア本体

【使い方】
  1. この zip を解凍し、フォルダごと任意の場所に置きます (デスクトップ等)
  2. M-Logger を USB ケーブルで PC に接続します
  3. update.bat をダブルクリックします
  4. 画面の指示に従い、Reset ボタンを押下したまま電源スイッチを ON にして
     赤 LED の点滅を確認します
  5. 何かキーを押すと自動的に書き込みが始まります (約 10 秒)
  6. 「書き込み完了」と表示されたら、電源を OFF → ON で新ファームウェアが
     動作します

【うまく行かない時】
  - 書き込みエラー → Reset 長押し電源 ON で赤 LED が点滅しているか確認
  - PC が認識しない → USB ケーブルを差し直して再度実行
  - 何度試しても失敗 → サポートに連絡

【注意】
  - 更新中は電源を切らないでください
  - 同時に複数の M-Logger を PC に接続しないでください
  - Windows 10 以降を想定しています
