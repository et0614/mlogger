# 詳細設定と常設モード

ここで扱う項目は通常の計測では使いません。
**設定を誤ると M-Logger の動作に支障が出る可能性があります**。

このうち「補正係数」「CO2 センサの校正・初期化」の 3 つは [計測の設定](settings.md) 画面でスマートフォンをシェイクしたときだけ表示されます。誤操作を防ぐため、シェイクという明示的な動作を介して初めて開放されるようになっています。

## M-Logger 名の変更

「M-Logger の名称を設定」から、スキャナ画面のリストに表示される名前を変更できます。
複数台運用する際の識別に使います。

## 補正係数

工場出荷時にキャリブレーション済みのため通常は触る必要はありません。
長期使用後にセンサ出力のドリフトを補正したい場合に、各センサごとの倍率・オフセットを線形に調整できます。

補正係数は v3 / v4 ともに **乾球温度・相対湿度・グローブ温度・照度・風速の 5 種類** にそれぞれ独立に設定します (CO2 は工場校正 + 自動自己校正のため対象外)。v4 で計測設定が 3 カテゴリに集約されていても、補正係数は引き続きセンサ単位です。

## CO2 センサの校正・リセット・完全初期化

CO2 センサ (Sensirion STCC4) には **自動自己校正 (ASC)** 機能があり、1 日に 1 回程度、屋外相当 (約 400 ppm) の新鮮空気に触れさせていれば、その濃度を基準に自動で校正されます。
通常の屋内計測でも、毎日窓開け・換気・屋外への持ち出しなどがあれば自動校正は機能するため、特別な操作は不要です。

長期間にわたって高 CO2 濃度の環境 (締め切った室内・温室・実験チャンバなど) で計測を続けたり、長期間使用後に出力ドリフトが気になる場合は、以下の手動操作で介入します。各操作は Sensirion datasheet (ICD01) の異なるコマンドに対応しています。

| ボタン | 動作 | 所要時間 | datasheet 該当 |
|--------|------|---------|---------------|
| **CO2 センサを校正する** | 既知 CO2 濃度を入力 → 30 秒測定 → 強制校正 | 約 35 秒 | §3.4.15 `perform_forced_recalibration` |
| **CO2 センサを工場リセット** | ASC / FRC 履歴を消去 + bypass phase 再開 (新品状態に戻す) | 約 90 ms | §3.4.11 `perform_factory_reset` |
| **CO2 センサを完全初期化** | 工場リセット → 12 時間連続運用 → 強制校正 (datasheet §1.1.4 "Initial Operation" を再現) | 約 12 時間 | 上記の組合せ |

### CO2 センサを校正する

明確に分かっている CO2 濃度の下にセンサを置き、その濃度を入力して校正します。

- 入力する CO2 濃度は、別の校正済み CO2 計または既知のガスで確認した値を使います
- 校正動作には約 **35 秒** の時間が必要です。その間、センサと周囲条件 (CO2 濃度・温度・湿度・電源) を安定させたまま放置してください

### CO2 センサを工場リセット

センサ内部に蓄積された ASC / FRC 履歴を消去して、購入直後と同じ状態に戻します。即時 (~90 ms) で完了します。

長期使用後にセンサ出力が明らかにおかしくなった場合 (ASC が誤った基準で収束してしまっている等) のリセット手段です。リセット後は datasheet §1.1.4 にある「Initial Operation」フェーズが再開され、12 時間の連続運用 + 新鮮空気への暴露で改めて精度を出していくことになります。

### CO2 センサを完全初期化

上記「工場リセット」+ 12 時間の連続安定化運用 + 指定 CO2 濃度での強制校正を **アプリ側で自動的にシーケンス実行** する操作です。
ユーザーが手動で 12 時間待つ必要はありませんが、その間 M-Logger は計測動作で占有されるので別の計測には使えません。指定 CO2 濃度 (400 ppm の新鮮空気が代表的) を維持できる環境で実行してください。

## PC との通信

PC + Zigbee で複数台同時運用する場合に使います。
このとき PC + XBee コーディネータが **親機**、各 M-Logger が **子機** となります。
具体的な接続手順とアドレス割り当ては PC 運用マニュアル (準備中) を参照してください。

## 常設モードへの移行

![常設モード移行ダイアログ](assets/screenshots/permanent_mode_dialog.png){ width="280" }

「常設モードへ移行」を選ぶと、M-Logger の電源を入れるたびに自動で計測を開始し、PC へデータ送信する設定になります。
壁面・天井等への据付運用を想定したモードです。

!!! warning "解除はスマートフォンからはできません"
    一度常設モードに移行すると、電源 OFF / ON でも解除されません。次節の Reset スイッチによる物理操作が必要です。

### 常設モードの解除

![Reset スイッチ位置 (旧版・暫定)](assets/screenshots/reset_switch_legacy.jpg){ width="280" }

!!! warning "画像は旧版"
    上記写真は旧ハードウェア (v3.x) の Reset スイッチ位置です。v4 ハードでは位置が異なります。撮影でき次第差し替えます。

本体の Reset スイッチを 3 秒以上長押しすると、常設モードが解除されて LED が 3 回点滅します。
